# Detail Page Rules

## Detail Page Priority

For finance detail pages:

1. completeness
2. readability
3. consistency
4. visual polish

## Fitting Order

When content does not fit:

1. reduce row padding
2. reduce font size
3. shorten headers
4. adjust column width
5. paginate

## Forbidden Moves

- dropping rows silently
- truncating important fields
- using unreadably tiny type to avoid pagination
- inconsistent units across pages

## Common Detail Page Types

- bank / channel detail
- project plan list
- responsibility list
- progress tracker
- appendix table

## Progress List Rules

- prefer row-based layout over decorative cards
- keep owner and amount close to project name
- preserve whitespace on the right if that improves readability
