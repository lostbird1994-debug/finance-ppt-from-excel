# Page Structures

## Default Finance Deck Structure

1. Cover
2. Total overview
3. Segment overview
4. Segment or channel detail pages
5. Next-month plan / key pipeline
6. Blockers and actions

## Financing Report Structure

1. Financing overview
2. Financing blocks / channels
3. National-bank detail
4. Local-bank detail
5. Leasing / supply-chain detail
6. Next-month plan projects
7. Blockers and actions

## Operating Review Structure

1. KPI overview
2. Segment summary
3. Variance page
4. Focus items
5. Actions

## Priority Rule

If pages must differ in design quality:

- A pages: cover / overview / segment overview / closing action page
- B pages: project and segment detail pages
- C pages: appendix-like overflow pages

A pages get most of the design effort.
